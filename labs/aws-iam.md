# Creating AWS IAM Accounts for labs

- Create a group : `students`
- Create IAM accounts like `student1@company.com` and add to group
- Atttach these policies to **group** (not individual accounts)
    - AmazonEC2FullAccess
    - AmazonS3FullAccess
    - AWSCloudShellFullAccess
    - IAMReadOnlyAccess
